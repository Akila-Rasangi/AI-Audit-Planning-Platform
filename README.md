# AI-Audit-Planning-Platform
Developed an AI-assisted audit planning platform to support audit teams in preparing structured audit plans. The system follows a six-stage workflow covering client setup, risk assessment, financial-ratio analysis, materiality calculation, AI-generated audit planning, manager approval, and audit-program generation.

• Designed and developed the frontend using React.js.

• Built backend APIs using Node.js and Express.js.

• Used MongoDB to store client details, audit data, risk responses, approval records, and generated outputs.

• Integrated Llama 3 through Ollama to generate audit-planning content.

• Implemented adaptive risk assessment and financial-ratio analysis workflows.

• Added materiality calculation and manager approval functionality.

• Supported generation of customized audit programs based on client-specific audit risks.
